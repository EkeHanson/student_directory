Student Directory · Build Instructions

Each student builds ONE profile page. Follow these steps to complete your page.
Your Mission: Create your personal profile page (yourname.html) that showcases your skills, projects, GitHub, LinkedIn, and more. Then, update the main directory so employers can find you.

Each student is assigned a unique HTML file. Use your name as the filename (e.g., adaobi.html, chidi.html).

If you don't have the template, copy the code from Section 7 below.

Customize Your Profile

Replace the placeholder content with your personal information. Here's what to update:

  • Photo: Replace your-photo.jpg with your actual image (or use a placeholder).
  • Name & Role: Update the <h1> and <p> in the hero section.
  • About Me: Write 2–3 sentences about yourself.
  • Skills: List technologies you've mastered (React, Python, AWS, etc.).
  • Projects: Add at least 2 projects with GitHub and live demo links.
  • Experience: Include internships, jobs, or relevant experience.
  • Education: Add appBrew Academy and any other education.
  • Social Links: Update GitHub, LinkedIn, Twitter, or portfolio URLs.
  • Resume: Link to your PDF resume (or add a placeholder).

Required Content Checklist

Must-Have

  • ✅ Profile photo
  • ✅ Full name & role
  • ✅ Bio / About Me
  • ✅ Skills list (5+)
  • ✅ 2+ projects with links
  • ✅ GitHub & LinkedIn links
  • ✅ Experience section
  • ✅ Education section

Nice-to-Have

  • ⭐ Resume download (PDF)
  • ⭐ Twitter / portfolio link
  • ⭐ Tech stack icons
  • ⭐ Project screenshots
  • ⭐ Testimonials
  • ⭐ Blog or articles

Update the Main Directory

Open index.html and find the students array (around line 180). Update your entry:

{ id: 1, name: "Your Full Name", role: "Your Role", photo: "your-photo.jpg", skills: ["Skill1", "Skill2", "Skill3"], bio: "Short bio (1 sentence)", linkedin: "https://linkedin.com/in/yourprofile", github: "https://github.com/yourusername", page: "yourname.html" // ← MAKE SURE THIS MATCHES YOUR FILE }
Important: The page value must exactly match your HTML filename (e.g., "adaobi.html"). This is how employers click through to your profile.

Test Your Page

  • Open index.html in your browser and click "View Full Profile" on your card.
  • Make sure your page loads correctly and all links work.
  • Check that your photo, skills, projects, and social links are visible.
  • Confirm the mobile menu works (resize your browser to test).

## 📝 Student Instructions

### Step 1: Claim Your Page

Each student is assigned a unique HTML file. **Use your name as the filename** (e.g., `adaobi.html`, `chidi.html`).

### Step 2: Copy the Profile Template

Start with the template provided below. It includes:
- Dark theme matching the main directory
- Navigation bar
- All required sections (About, Skills, Projects, Experience, Education)
- Social links and resume download button

### Step 3: Customize Your Profile

Replace the placeholder content with your personal information:

#### Required Updates:
- **Photo:** Replace `your-photo.jpg` with your actual image
- **Name & Role:** Update the hero section `<h1>` and `<p>`
- **About Me:** Write 2–3 sentences about yourself
- **Skills:** List technologies you've mastered (React, Python, AWS, etc.)
- **Projects:** Add at least 2 projects with GitHub and live demo links
- **Experience:** Include internships, jobs, or relevant experience
- **Education:** Add appBrew Academy and any other education
- **Social Links:** Update GitHub, LinkedIn, Twitter, or portfolio URLs
- **Resume:** Link to your PDF resume (or add a placeholder)

#### Required Content Checklist:
- ✅ Profile photo
- ✅ Full name & role
- ✅ Bio / About Me
- ✅ Skills list (5+ technologies)
- ✅ 2+ projects with links
- ✅ GitHub & LinkedIn links
- ✅ Experience section
- ✅ Education section

#### Nice-to-Have Additions:
- ⭐ Resume download (PDF)
- ⭐ Twitter / portfolio link
- ⭐ Tech stack icons
- ⭐ Project screenshots
- ⭐ Testimonials
- ⭐ Blog or articles

### Step 4: Update the Main Directory

Open `index.html` and find the `students` array. Update **YOUR** entry:

```javascript
{
  id: 1,
  name: "Your Full Name",
  role: "Your Role",
  photo: "your-photo.jpg",
  skills: ["Skill1", "Skill2", "Skill3"],
  bio: "Short bio (1 sentence)",
  linkedin: "https://linkedin.com/in/yourprofile",
  github: "https://github.com/yourusername",
  page: "yourname.html"  // ← MAKE SURE THIS MATCHES YOUR FILE
}
